GODADDY DNS for themaniak.online (Render hosting)
=================================================

Your Laravel app runs on Render — GoDaddy only points the domain.

Full deploy guide: deploy/render-setup.txt (Part 3 + Part 4)

Quick steps
-----------
1. Render → themaniak-games → Settings → Custom Domains
2. Add themaniak.online and www.themaniak.online
3. GoDaddy → Domain → DNS → add records Render shows, usually:

   Type    Name    Value
   A       @       (IP from Render)
   CNAME   www     themaniak-games.onrender.com

4. Wait up to 1 hour for DNS, then test https://themaniak.online/up

Remove old Website Builder A/CNAME records if they conflict.

Need help? Screenshot GoDaddy DNS + Render custom domain page.
