THE MANIAK — Replace shop on themaniak.online (cPanel / shared hosting)
=======================================================================

1. BACK UP the current shop (download files + export any shop DB).

2. REMOVE or disconnect the old shop app from the domain:
   - Domains → themaniak.online → document root was probably public_html
   - Delete old shop files from public_html AFTER backup (or rename folder to shop_backup)

3. UPLOAD this Laravel project ABOVE public_html, e.g.:
   /home/YOURUSER/themaniak-app/     ← full project (app, bootstrap, config, …)
   /home/YOURUSER/public_html/       ← ONLY contents of my-laravel-app/public/

   Easier layout (recommended):
   /home/YOURUSER/themaniak-app/          ← entire repo
   public_html → symlink to themaniak-app/public
   (In cPanel File Manager: remove public_html, create symlink, or use SSH:
    rm -rf public_html && ln -s themaniak-app/public public_html)

4. CREATE MySQL database in cPanel:
   - Database: themaniak_games
   - User + password with ALL privileges on that DB

5. ENV on server:
   - Copy deploy/env.production to themaniak-app/.env
   - Use SAME Aiven/cloud DB credentials as local .env
   - Save Aiven CA cert to database/ssl/ca.pem on server
   - Run in Terminal (cPanel → Terminal):
     cd ~/themaniak-app
     php artisan key:generate
     php artisan migrate --force
     php artisan storage:link
     php artisan config:cache
     php artisan route:cache

6. PHP version: 8.2+ (Select PHP Version in cPanel)

7. SSL: cPanel → SSL/TLS → AutoSSL or Let's Encrypt for themaniak.online

8. DNS: A record @ → server IP, www → server IP (or CNAME to themaniak.online)

9. GOOGLE LOGIN (optional): Google Cloud Console → OAuth redirect:
   https://themaniak.online/auth/google/callback

10. TEST: https://themaniak.online/up  should return {"status":"ok"}
